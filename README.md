# AI Skills

A collection of reusable AI skills designed to automate workflows, improve productivity, and standardize common tasks across projects.

## Overview

This repository contains structured AI skills that can be used with tools like `npx skills`, OpenCode, or any compatible prompt-based system.

Each skill is designed to be:
- Reusable
- Minimal
- Easy to integrate
- Focused on a specific task (e.g., PR review, commit generation, code analysis)

## Structure

Each skill follows this structure:


skill-name/
├── skill.json # Metadata and configuration
└── prompt.md # Core logic (prompt)


### Example


pr-review/
├── skill.json
└── prompt.md
